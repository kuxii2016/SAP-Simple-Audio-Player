# SAP (Simple Audio Player)

A lightweight and easy-to-use audio player built with **C#** and **WPF**, designed to provide a simple music listening experience without unnecessary complexity.

## ✨ Features

- 🎵 Audio playback with a clean and intuitive interface
- 📋 Playlist support
- 🎚 Built-in Equalizer
- 🎮 Windows media key and multimedia button support
- 💬 Discord Rich Presence (RPC) integration
- ⚡ Lightweight and responsive
- 🖥 Modern WPF user interface

## 🎯 Goals

SAP focuses on simplicity, performance, and ease of use. The application provides the essential features users expect from a music player while keeping the interface clean and straightforward.

## 🛠 Built With

- C#
- WPF (.NET)

## 🚀 Getting Started

1. Download the latest release.
2. Launch `SAP.exe`.
3. Add your music files to a playlist.
4. Enjoy your music.
